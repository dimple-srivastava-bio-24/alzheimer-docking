# alzheimer-docking
This project demonstrates molecular docking of Donepezil with acetylcholinesterase using AutoDock Vina. The study evaluates binding affinity and visualizes protein–ligand interactions to understand the inhibitory mechanism relevant to Alzheimer’s disease.

---

🖤 Overview

This project focuses on the molecular docking analysis of Donepezil with acetylcholinesterase, a key target in Alzheimer’s disease.
The goal is to evaluate binding affinity and visualize protein–ligand interactions using computational tools.

---

🧪 Tools & Technologies

- AutoDock Vina
- PyMOL
- Open Babel
- Ubuntu (Virtual Environment)

---

🧬 Biological Background

Alzheimer’s disease is associated with reduced acetylcholine levels in the brain.
Donepezil acts as an acetylcholinesterase inhibitor, preventing the breakdown of acetylcholine and improving neural transmission.

---

⚙️ Methodology

- Protein retrieved from Protein Data Bank (PDB ID: 4EY7)
- Ligand retrieved from PubChem
- Protein prepared by removing water and adding hydrogens
- Ligand optimized and converted into pdbqt format
- Grid box defined around active site
- Docking performed using AutoDock Vina
- Interactions visualized using PyMOL

---

📊 Results

✨ Binding Affinity: -8.921 kcal/mol

✔ Strong interaction observed
✔ Ligand fits well in active site
✔ Stable protein–ligand complex formed
